Setup Guide

Step-by-step integration of WPForms, Google Sheets, Zapier, and Google Calendar for the Urban Niche Academy.

1
WPForms Application Form
Create a new form on WPForms with fields for collecting candidate information and embed it on your careers page.
  • Create new form in WPForms
  • Add field: Full Name
  • Add field: Email
  • Add field: Phone
  • Add field: Why do you want to join Urban Niche Co? (textarea)
  • Add field: Previous sales experience? (dropdown: None, 1-2 years, 3-5 years, 5+ years)
  • Add field: How did you hear about us? (dropdown)
  • Embed form on urbannicheco.com/careers page
2
Google Sheet Setup
Create the master Google Sheet that will store all candidate data and assessment results.
  • Go to docs.google.com/spreadsheets
  • Create new spreadsheet: "Urban Niche Academy"
  • Create tab: "Candidates"
  • Create tab: "Assessments"
  • Create tab: "Training"
  • Create tab: "Config"
  • In Config tab, add: CALENDAR_BOOKING_URL, PASSING_THRESHOLD, etc.
  • Run setupSheets() from Google Apps Script
3
Zapier Connection: WPForms to Google Sheets
Automatically sync form submissions to your Google Sheet.
  • Log into Zapier.com
  • Create new Zap
  • Trigger: WPForms — New Form Entry
  • Select your application form
  • Action: Google Sheets — Create Spreadsheet Row
  • Map Name → Column B
  • Map Email → Column C
  • Map Phone → Column D
  • Auto-generate ID in Column A
  • Set Stage to "Applied" in Column F
  • Test and turn on Zap
4
Zapier Connection: Google Sheets to Gmail
Send automated confirmation emails when new candidates are added.
  • Create new Zap
  • Trigger: Google Sheets — New Row in Candidates
  • Select your spreadsheet and Candidates tab
  • Action: Gmail — Send Email
  • From: your@urbannicheco.com
  • To: {{Email}}
  • Subject: "Welcome to Urban Niche Academy!"
  • Body: Application confirmation with assessment link
  • Test and turn on Zap
5
Google Calendar Appointment Scheduling
Set up interview scheduling so candidates can book time with your team.
  • Go to calendar.google.com
  • Click Settings → Appointment Schedules
  • Create schedule: "Urban Niche Co. Interview"
  • Duration: 30 minutes
  • Set available hours (e.g., 9 AM - 5 PM)
  • Set available days (Mon-Fri)
  • Timezone: Your timezone
  • Copy the booking link
  • Paste into CALENDAR_BOOKING_URL in the app config
